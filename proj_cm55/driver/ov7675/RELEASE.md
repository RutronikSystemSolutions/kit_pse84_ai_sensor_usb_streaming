# OV7675 DVP Camera driver library release notes

This driver library provides functions for supporting [OV7675 DVP Camera module](https://www.arducam.com/640x480-0-3-mp-mega-pixel-lens-ov7675-cmos-camera-module-with-adapter-board.html) from OmniVision on PSOC&trade; Edge device.


## What's included?

API for initializing the driver and storing camera data in buffer. <br> Refer to the [README.md](./README.md) and the [API Reference Guide](./api_reference.md) for a complete description of the camera-dvp-ov7675.
